8.
    q: If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
    a: Java doesn't allow you to overload function with the same signature (same number of args, same type of arg respectively). If you overload constructor (String title) by a constructor (String category), it will cause an error.
    Noted that function with the same number or args and same type of each arg but with different order, it will have different signature, which means overload-able.
